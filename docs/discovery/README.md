# Discovery

진행 중인 탐색, raw Q&A, product research, 임시 분석을 이 디렉터리에 둔다.

Discovery note는 current implementation authority가 아니다.

규칙:

- Discovery가 질문을 만들면 `07_QUESTIONS_REGISTER.md`에 추가한다.
- Discovery가 accepted lightweight decision을 만들면 `08_DECISION_REGISTER.md`에 추가한다.
- Discovery가 major architecture decision을 만들면 `docs/adr/` 아래 ADR을 추가한다.
- Product scope가 바뀌면 `01_PRD.md`를 갱신한다.
- Roadmap / status가 바뀌면 `docs/04_IMPLEMENTATION_PLAN.md`를 갱신한다.
- Active current position이 바뀌면 `docs/context/current-state.md`를 갱신한다.
