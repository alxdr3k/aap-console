# Generated Docs

이 디렉터리는 code, schema, migrations, config, specs에서 파생된 문서를 둔다.

**Generated docs는 손으로 편집하지 않는다.** 내용이 틀렸다면 output이 아니라
generator 또는 source를 수정한다.
