# Archived Design Notes

Archived design note는 과거 탐색과 이전 가정을 보존한다.

이 문서들은 current implementation authority가 아니다.

Archived design을 구현에 사용하기 전에는 다음을 먼저 확인한다.

1. `docs/context/current-state.md`
2. `docs/04_IMPLEMENTATION_PLAN.md`
3. `docs/01_PRD.md`
4. `docs/02_HLD.md`
5. 관련 ADR
6. 구현이 존재하는 영역은 current code/tests/migrations
