# AAP Console — Legacy Implementation Status

이 문서는 기존 구현 상태 매트릭스의 compatibility pointer다.

Canonical status는 다음 문서로 이동했다.

- Roadmap / status ledger: [`04_IMPLEMENTATION_PLAN.md`](./04_IMPLEMENTATION_PLAN.md)
- Active session summary: [`context/current-state.md`](./context/current-state.md)
- Acceptance gates: [`06_ACCEPTANCE_TESTS.md`](./06_ACCEPTANCE_TESTS.md)
- Current implementation navigation: [`current/`](./current/)

기존 matrix의 완료/부분/미구현 상태는 boilerplate `24851cf`의 taxonomy에 맞춰
implementation status (`landed`, `accepted`, `planned` 등)와 gate status
(`defined`, `not_run`, `passing` 등)로 분리했다.
